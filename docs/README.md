# The Grackle Server Platform

**Grackle** is a collection of Bash scripts designed to make server management simple &amp; approachable. It aims to adhere to the [UNIX Philosophy](https://bestpoint.institute/tactics/unix-philosophy).

Built for hobbyists and powerusers. __"cPanel for Commodore 64"__ if you will, only it runs in a modern Debian environment with enterprise-grade packages.

This software is in a highly-experimental "pre-beta" phase of sorts, so use at your own peril.
